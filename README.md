# দোকান টুলস প্রো (Dokan Tools Pro)

AI-powered ID Card Sheet Generator (Module 1) and Passport Photo Maker (Module 2) — Xerox দোকানের জন্য তৈরি একটি প্রোডাকশন-রেডি ওয়েব অ্যাপ্লিকেশন।

## Tech Stack

- Vanilla JavaScript + Vite (no framework)
- OpenCV.js
- ONNX Runtime Web
- pdf-lib
- 100% client-side (browser-based), deployed on Cloudflare Pages

## How to Run

```bash
npm install
npm run dev
```
